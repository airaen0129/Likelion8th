Github Command

Push to empty repository

git init
git remote add origin 레포지토리 주소
git add .
git commit -m "(원하는 내용)"
git push origin master

Push to exist repository

git clone 레포지토리 주소
git add .
git commit -m "(원하는 내용)"
git push origin master
# [fetch, pull 필요할 경우 Pull from exist repository 참고]

Pull from exist repository

git fetch
git pull
# [충돌 발생 시 충돌 해결]

연결된 이후 push

# fetch, pull, merge 등 수행
git commit -m "(원하는 내용)"
git push origin master
